
# EC2- Elastic Compute Cloud

### Instance Types:
1. General Purpose  ( t series )
2. Compute optimized ( c series )
3. Memory Optimized ( r series )
4. Storage Optimized ( i series )
5. Accelerated Computing ( p series )


### Purchasing Options:
1. On Demand
2. Reserved
3. Dedicated Host
4. Spot Instance
